## Overview

The GA Public Service System, also known as the Integrated Security Gate System, is a system developed to manage and monitor entry and exit access at the PT Ajinomoto Indonesia Mojokerto Factory compound. In its first phase, the system focused on real-time recording and management of employee access (counting in/out) using RFID/QR technology integrated with automated gate hardware. Beyond simply opening and closing the gate, the system also performs access validation, transaction log recording, device status monitoring, and anomaly detection, such as mismatches between entry and exit records.

This was a client project for PT Greatsoft Solusi Indonesia, developed together with one senior developer during my internship there.

## Key Features

- **RFID/QR Access Validation**: Validates employee access using RFID and QR-based credentials at the gate.
- **Real-Time Gate Monitoring**: Live monitoring of gate and device status across the factory compound.
- **Transaction Logs**: Detailed logging of every access transaction for audit and tracking purposes.
- **Anomaly Detection**: Automatic detection of access anomalies, such as inconsistencies between entry and exit counts.
- **Emergency Access**: A dedicated flow for handling emergency access scenarios.

## My Role and Contributions

I held a double role on this project. I first designed the complete UI/UX for both the web and mobile versions of the system independently, starting from a blank slate with no existing prototype to reference. After completing the design phase, I then moved into implementation, slicing and coding the mobile application together with one senior developer using Flutter. My implementation work covered the login flow, gate monitoring, emergency access, access logs, and the profile page.

## Challenges and Learnings

Owning both the design and the mobile implementation within a single project made it clear that the two cannot be treated as separate disciplines. Every decision made on the design side directly affected how the feature needed to be implemented, and vice versa, which reinforced the importance of designing with implementation feasibility in mind from the start.

## Outcome

The system delivered a functioning real-time access management solution for the factory compound, improving visibility into employee access patterns and enabling faster detection of security anomalies.
